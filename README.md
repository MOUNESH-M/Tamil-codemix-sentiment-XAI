\# Tamil Code-Mixed Sentiment Analysis with Explainability (XAI)



This is a research sprint project focusing on:

\- Low-resource Tamil code-mixed sentiment analysis

\- Transformer-based classification (IndicBERT/MuRIL)

\- Explainable AI (LIME + SHAP)

\- Real-world social media Tamil-English data

\- Publishable results + clean Git workflow



\## Project Structure

\- `data/` : Dataset (ignored from Git)

\- `src/` : Python scripts

\- `notebooks/` : Colab/Jupyter workflows

\- `paper/` : Research paper draft

\- `figures/` : Plots, SHAP/LIME outputs

\- `outputs/` : Models, logs
